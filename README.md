# Movie_Rental_Analysis
The project aims to uncover patterns and trends in rental performance, customer behavior, staff productivity, and film popularity across different store locations and time periods. It provides a data-driven foundation for improving business strategies in marketing, inventory management, customer retention, and operational efficiency.

## ğŸ“ Project Overview

This project presents a comprehensive **Movie Rental Analysis Dashboard** for a video rental business using the well-known **Sakila sample database**. The goal is to extract meaningful insights about customer behavior, film performance, inventory distribution, staff productivity, and operational efficiency.

The analysis was conducted using **SQL for data modeling and transformation**, **Excel for early-stage EDA**, and **Power BI** to build an interactive, visual dashboard. The project is structured using the **MECE framework** to ensure that all aspects of the business are explored clearly and without overlap. It includes a **dashboard file**, **PowerPoint slides**, and a **project report** to summarize findings.
